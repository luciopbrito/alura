# desenhando processos

É necessário observar o fluxo com foco em abstrair ações importantes para a realização da tarefa. Mapear um fluxo requer
uma análise dos casos mais relevante para que aquele processo seja finalizado; Reconhecer os pontos de atenção durante o
processo é a chave para desenhar processos.

- Definir pontos de ruptura claros (divisões para sub-processos)
- Criar perguntas binárias (sim e não)

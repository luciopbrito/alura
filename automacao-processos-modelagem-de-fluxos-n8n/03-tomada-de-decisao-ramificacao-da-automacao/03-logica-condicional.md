# conditional logic

Durante a definição do fluxo do processo, existiram momentos que o fluxo deve decidir qual linha de resolução será
seguida, e para isso se utiliza condicionais com foco em responder perguntas simples.

O N8N permite resolver essas condicionais com um nó condicional, com os seguinte passos:

1. Determina uma entrada.
2. Determina condicional.

Ao final, será possível definir as duas situações de maneira isoladas.

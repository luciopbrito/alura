# *Human in the loop*

Conforme a criação das automações, existem processos que ainda exigem a presença de algum ser humano, esses recurso para
o N8N é viabilizado pelo nó *human in the loop*.

O N8N possibilita diversas formas para recuperar a resposta pelo ser humano dentro do processo, sendo uma delas via
e-mail. Isto por ser feito por: confirmação com aprovações; formulários customizados; confirmação do recebimento do
e-mail.

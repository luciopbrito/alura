# Triggers

N8N oferece milhares de gatilhos para iniciar uma automação. Entretanto para dar início é necessário fazer uma
credencial de acesso para a plataforma que deseja escutar os gatilhos.

## Credenciais

As credenciais podem ser feitas via SSO (Single Sign-On) ou [OAuth2](#oauth).

## Referências
